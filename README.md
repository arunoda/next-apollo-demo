# next-apollo-demo
Simple Next.js example with Apollo 

To run the app locally
1). Go to server folder
   1) npm install
   2) npm start
   3) Server will be ready at: http://localhost:4000/  

2) Go to client folder
   1) npm install
   2) npm build
   3) npm start
   4) Application will be running at: http://localhost:3000
